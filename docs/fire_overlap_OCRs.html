#-----------------------------
# Helper: BED (0-based start, end exclusive or inclusive doesn't matter much if consistent)
# We convert to 1-based inclusive GRanges: start = start0+1, end = end
#-----------------------------
bed_to_gr <- function(dt, keep_chr = NULL) {
  stopifnot(ncol(dt) >= 3)
  dt <- copy(dt)
  setnames(dt, 1:3, c("chr","start0","end"))
  gr <- GRanges(
    seqnames = dt$chr,
    ranges   = IRanges(start = dt$start0 + 1L, end = dt$end)
  )
  if (!is.null(keep_chr)) gr <- gr[seqnames(gr) %in% keep_chr]
  gr
}

Introduction

To evaluate how FIRE peaks relate to other OCR peaks (), we performed two overlap analyses: base-pair overlap and peak-level overlap.

Data

Data: https://xsun1229.github.io/fiberseq/data.html

For DNase/ ATAC-seq/ cCRE we used GM12878 cell line

1. Base-pair (bp) overlap

FIRE peaks and the comparison peak set were first converted to GRanges objects using hg38 genomic coordinates (BED format converted from 0-based to 1-based indexing).

To avoid double counting genomic coverage, overlapping peaks within each dataset were merged using reduce(), producing non-overlapping genomic intervals.

The shared genomic coverage between the two datasets was then calculated using:

  • intersect() to obtain overlapping genomic segments
  • width() to compute the number of overlapping base pairs

From this analysis we report:

  • total base pairs covered by FIRE peaks
  • total base pairs covered by the comparison peak set
  • overlapping base pairs

This analysis measures how much genomic space is shared between the two peak sets.

2. Peak-level overlap

To assess overlap at the individual peak level, we performed overlap detection using the original (non-merged) peaks.

Peak overlaps were identified using findOverlaps() with a minimum overlap threshold of 100 bp, meaning two peaks were considered overlapping only if they shared ≥100 bp.

From this analysis we report:

  • total number of FIRE peaks
  • total number of comparison peaks
  • number of FIRE peaks overlapping comparison peaks
  • number of comparison peaks overlapping FIRE peaks
  • total number of overlapping peak pairs

Peak-level overlap was summarized using a two-set Venn diagram, showing:

  • FIRE peaks only
  • comparison peaks only
  • overlapping peaks

Because genomic intervals can overlap in a many-to-many manner, the Venn intersection was defined as the number of FIRE peaks that overlap the comparison peak set by ≥100 bp.
